# qv scripts

Private qvOS helper scripts live here.

Use this namespace for scripts called by qvOS Hyprland, Waybar, branding, or system config. Only promote a script to `bin/` when it should be a public Omarchy command.

Layout:

```text
qv/scripts/
  branding/
  defaults/
  hyprland/
  screensaver/
  tui/
  waybar/
```
